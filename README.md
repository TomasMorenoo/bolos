# 🎳 Bolos — Sistema de Anotación de Bowling

App web para registrar partidas de bowling con amigos, llevar el historial completo y consultar estadísticas detalladas de cada jugador. Desarrollada con **Python y Flask**, base de datos **SQLite**, hosteada en VPS con **Docker**.

---

## 💡 Origen del proyecto

Nació de una necesidad real jugando al bowling con amigos: queríamos saber quién tiraba mejor, cuántos strikes acumulaba cada uno y tener un historial de todas las partidas. En vez de anotarlo en papel, armé esto.

---

## ✨ Funcionalidades

### 🎮 Registro de Partidas
- Soporte para **múltiples jugadores** por partida.
- Registro de puntajes, strikes y spares por jugador.
- Historial completo de todas las partidas jugadas.

### 📊 Dashboard de Estadísticas
- ⚡ **Total de Strikes acumulados** por jugador.
- 🔋 **Total de Spares acumulados** por jugador.
- 🔥 **Récord de Strikes en una sola partida** por jugador.
- ⚒️ **Récord de Spares en una sola partida** por jugador.
- 🏆 **Top 5 histórico** — las mejores puntuaciones de todos los tiempos con fecha y jugador.
- 📈 **Promedios** de puntos por jugador.
